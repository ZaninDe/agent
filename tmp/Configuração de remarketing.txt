Se você já utiliza listas de remarketing para anúncios da Rede de Pesquisa  (RLSA, na sigla em inglês),

Tag do Google 
Analytics (GA) Se você deseja usar dados analíticos detalhados do site para o remarketing, a tag do GA é recomendada. 
O GA também fornece recursos completos de dados analíticos de website para analisar e otimizar 
o desempenho. 
 
Se você já utiliza o GA, poderá ativar o remarketing facilmente  a partir da UI sem atualizações de tag. 
Nas configurações avançadas, defina Ativar recursos de anunciante  como ATIVADO . Crie listas de remarketing 
com base em URLs de página da Web, parâmetros personalizados ou no comportamento do website de seus 
clientes, como tempo no site ou número de páginas acessadas. 
Gerenciador de tags 
do Google (GTM) Se você deseja inserir uma tag totalmente nova em seu website, o Gerenciador de tags do Google (GTM, 
na sigla em inglês) é recomendado. É uma tag única para todo o site que usa regras para disparar tags do Google 
e de terceiros. Insira a tag de contêiner em todas as páginas de seu site. 
 
Para implementar o Gerenciador de tags do Google , copie o snippet de código fornecido na interface do 
Gerenciador de tags, substituindo GTM-XXXX  pelo ID de seu contêiner. Cole este snippet na página de 
modelo de seu website para que ele apareça imediatamente depois da tag <body> de abertura. Crie listas de remarketing separadas para usuários da página inicial, usuários da página 
de categoria, usuários da página de produto ou oferta, usuários que abandonaram carrinhos 
e usuários antigos que já converteram. Tudo o que contribuir para simplificar.

Como configurar o remarketing 2/2 Combine o remarketing com o lance  de CPA desejado 
O lance  de CPA desejado oferece uma maneira mais inteligente, fácil e eficaz para comprar na 
Rede de Display. Ele ajuda você a inserir seus anúncios onde há mais conversões.  
Use todos os tipos de anúncios gráficos, inclusive responsivos, para dispositivos móveis, 
de imagem e HTML5 
Os anúncios funcionam melhor quando são personalizados de acordo com o cliente que você 
deseja alcançar. Use todos os tipos de tamanhos e formatos de anúncio, e você terá mais chances 
de aparecer em mais lugares na hora certa.
